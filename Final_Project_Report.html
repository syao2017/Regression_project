Problem

Problem Statement

The objective of this statistical report is to investigate the factors that significantly influence Raptor ratings in the context of NBA player performance. Raptor scores are numerical metrics that quantify a players contribution to both team offense and team defense per 100 possessions, relative to a league-average player. This study aims to understand the relationships between the overall Raptor rating and various player statistics, such as position, assist percent, turnover rate, rebound rate, steal percentage, blocking percentage, and age.

The motivation behind this analysis is to gain insights into the attributes that differentiate a valuable basketball player in the NBA. By comprehensively examining the impact of different player statistics on Raptor ratings, this study aims to shed light on what qualities contribute most significantly to a players overall performance. Furthermore, this research seeks to determine whether it is possible to predict a players value based on statistical metrics and identify which statistics are pivotal in defining a players significance to the league.

Research Questions

  1. To explore the correlation between Raptor ratings and various player statistics, including position, assist percent, turnover rate, rebound rate, steal percentage, blocking percentage, and minutes per game.
  2. To identify which player statistics exhibit the strongest influence on Raptor ratings and contribute significantly to a players overall performance evaluation.
  3. To uncover potential statistical indicators that may not strongly affect Raptor ratings but still hold importance in the broader context of basketball game play.
  4. To assess the feasibility of predicting a players value to the NBA based on their performance statistics, particularly focusing on Raptor ratings and other relevant attributes.

Cross Variance

I did 2 cross variance one with the Atlanta data, and the other with the 2020 NBA data to see if the model was able to perform not only with the Atlanta data but also with the 2020 NBA data. In both cases the data performed well the RMSE of the Atlanta data was 4.01 which is relatively high. But the NBA data was very interesting, with a RMSE of 7.82 which was incredibly surprising since the model was used from just a single team, indicating some relative similarities in what teams looked for atleast during the 2020 NBA season.

# Load the caret package
library(caret)
## Loading required package: ggplot2
## Loading required package: lattice
# Set the number of folds
num_folds <- 5

# Create a data partition for cross-validation
set.seed(123)  # for reproducibility
folds <- createFolds(nba$`Raptor+/-`, k = num_folds)

# Perform k-fold cross-validation
results <- lapply(folds, function(fold_indices) {
  train_data <- nba[-fold_indices, ]
  test_data <- nba[fold_indices, ]
  
  # Build and train your model on the training data
  model <- lm(nba$`Raptor+/-` ~ nba$`AST%` + nba$`BLK%`)
  
  # Make predictions on the test data
  predictions <- predict(model, newdata = test_data)
  
  # Calculate evaluation metrics (e.g., RMSE)
  rmse <- sqrt(mean((test_data$`Raptor+/-` - predictions)^2))
  
  return(rmse)
})

# Calculate the mean and standard deviation of the RMSE values
mean_rmse <- mean(unlist(results))
sd_rmse <- sd(unlist(results))

cat("Mean RMSE:", mean_rmse, "\n")
## Mean RMSE: 7.816871
cat("Standard Deviation RMSE:", sd_rmse, "\n")
